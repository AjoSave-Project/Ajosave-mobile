# Gilroy Font Installation

## Required Font Files

To complete the onboarding screen design, you need to download and place the following Gilroy font files in this directory:

1. **Gilroy-Regular.ttf**
2. **Gilroy-Medium.ttf**
3. **Gilroy-SemiBold.ttf**
4. **Gilroy-Bold.ttf**

## Download Instructions

### Option 1: Font Marketplaces (Recommended)

Gilroy is a premium font available from various font marketplaces:

1. **MyFonts**: [MyFonts - Gilroy](https://www.myfonts.com/collections/gilroy-font-radomir-tinkov)
2. **Creative Market**: Search for "Gilroy Font"
3. **FontSpring**: Search for "Gilroy"

After purchasing:
1. Download the font family
2. Extract the ZIP file
3. Copy the following files from the extracted folder to this directory:
   - `Gilroy-Regular.ttf`
   - `Gilroy-Medium.ttf`
   - `Gilroy-SemiBold.ttf`
   - `Gilroy-Bold.ttf`

### Option 2: Free Alternatives

If you need a free alternative, consider similar geometric sans-serif fonts:
- **Montserrat** (Google Fonts)
- **Nunito Sans** (Google Fonts)
- **Work Sans** (Google Fonts)

Note: Gilroy is a commercial font and requires a license for use.

## After Installation

Once the font files are placed in this directory:

1. The fonts will be automatically loaded by Expo
2. The onboarding screen will use Gilroy fonts as specified in the design
3. If fonts are not found, the app will fall back to system fonts

## Verification

After adding the fonts, run:
```bash
npx expo start --clear
```

This ensures the fonts are properly loaded in your app.
